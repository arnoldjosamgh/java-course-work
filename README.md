# Java Course Work

This repository contains solutions for the Java course assignment.

## GitHub Repository
[https://github.com/arnoldjosamgh/java-course-work](https://github.com/arnoldjosamgh/java-course-work)

## Files Included

1.  **QN1_Answers.java**:
    *   **Question 1 (Part A)**: A program to compute grades for 5 students based on their scores. It determines the grade (1-9) and remark (D1, D2, etc.) for each score and prints a summary of grades afterwards.

2.  **QN1_PartB_Answers.java**:
    *   **Question 1 (Part B)**: A program to determine the winner of a vehicle auction. It accepts bids from 3 bidders, identifies the highest bid, and calculates if a profit or loss was made based on the vehicle cost and expenses.

3.  **QN2_Answers.java**:
    *   **Question 2**: A GUI (Graphical User Interface) application for a "New Student Registration Form". It includes fields for name, email, password, DOB, gender, and department, and validates the input before saving successful registrations to a `students.csv` file.

## How to Run

Compile the files:
```bash
javac QN1_Answers.java QN1_PartB_Answers.java QN2_Answers.java
```

Run Question 1:
```bash
java QN1_Answers
```

Run Question 1 Part B:
```bash
java QN1_PartB_Answers
```

Run Question 2:
```bash
java QN2_Answers
```
